# BeatSaberQuestOneClick

App that registers `beatsaver://` and `web+bsmap://` URIs letting you download maps from BeatSaver using the OneClick link feature directly through the Quest browser.  

## Install

Go to the [Releases](https://github.com/counter185/BeatSaberQuestOneClick/releases) tab, download the latest APK and sideload it onto your Quest using ADB.

## Usage

Open a web browser on your Quest (does not have to be the stock browser) and go to a website that lets you download Beat Saber maps using the one-click download feature, like BeatSaver or BeatLeader.

Click the one-click download button to instantly download and extract the map to `/sdcard/ModData/com.beatgames.beatsaber/Mods/SongCore/CustomLevels/`. The first time you do this, you will be prompted to grant file manager permissions so that the app can access this folder.
